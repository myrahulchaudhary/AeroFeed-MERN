Project Overview
AeroFeed is a full-stack MERN application where users can:
Register & Login securely
Create and manage posts
Generate and share referral links
Track referral-based user registrations
Maintain authenticated sessions using JWT
This project demonstrates full-stack development skills including backend API development, database management, authentication, and frontend integration.
